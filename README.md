## Objetive

Create a web server, should be able to works with some services as Wordpress, PhpMyAdmin, and a database.

## Use

Using docker:<br/>
`make build` - Creates the image 'mrubio:LEMP'.<br/>
`make run` - Creates and init container.<br/>
Allways localhost.
